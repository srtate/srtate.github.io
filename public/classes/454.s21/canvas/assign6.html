  4. Explore this concept by creating simple graphs that satisfy each of the stated properties below (each one has a solution that is just a 4-vertex graph).   5. The input to the traveling-salesperson problem is a complete weighted graph, which is difficult to draw when there are more than 5 vertices.   6. Some NP-complete problems that take numbers in their input become easy to solve on restricted ranges of numbers, and some remain NP-complete.

    1. Consider this slightly modified version of the set partition problem: A weighted set of items S is a set of distinct items S = {x1, x2, …, xn} along with a weight function so that we can refer to the weight of item xi as w(xi). Define

      SET-PARTITION  = {⟨S⟩ : S is a weighted set, and there is a subset A ⊆ S such that ∑x ∈ Aw(x) = ∑x ∈ S − Aw(x)}.

      If weights are arbitrary, then this is equivalent to the standard set partition problem, so is NP-complete. However, if all of the weights are either 1 or 2, then the problem is easily solvable in polynomial time. Describe an algorithm that solves this problem in this special setting. Your description can be brief, but it does need to be clear and unambiguous. You can use pseudocode or an English description of the algorithm, and you should include a brief explanation of why it works.

    2. Consider the TSP problem when all weights are 1 or 2. Even with this restriction on the weights, the problem is NP-complete. Using the NP-completeness proof of TSP as a guide (on pages 1096-1097 of the textbook) describe why this is so. You do not have to give a full NP-completeness proof, but you should explain how this setting modifies the book’s proof and why it is still valid.

    3. Do complete graphs in which all weights are 1 or 2 satisfy the triangle inequality? Give a one sentence explanation of why.

    4. Is the Approx-TSP-Tour algorithm from the book useful for this kind of TSP problem, where all weights are 1 or 2? Give a brief one or two sentence explanation of your answer.
