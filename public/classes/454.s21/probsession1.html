Problem Session 1 - March 16, 2020

Today’s Problem:

A “palindrome partition” of a string is a subdivision of the string into substrings that are all palindromes. For example, consider the string \(\texttt{CACACDCABDB}\). Since the first and last letters of the string are different, it’s obviously not a palindrome, but it does contain palindromes as substrings. Here is a palindrome partition for this string:

\[ \texttt{CAC}\ |\ \texttt{ACDCA}\ |\ \texttt{BDB} \]

Does every string have a palindrome partition? For example, \(\texttt{ABCDAB}\) doesn’t contain any non-trivial palindromes, so can we partition it into palindrome substrings?

Yes! Since every one-character string is a palindrome, the following is a legitimate palindrome partition (and the only palindrome partition for this string):

\[ \texttt{A}\ |\ \texttt{B}\ |\ \texttt{C}\ |\ \texttt{D}\ |\ \texttt{A}\ |\ \texttt{B} \]

A single string can have many different palindrome partitions. In the extreme case, think about how many palindrome partitions the string \(\texttt{AAAAA}\cdots\texttt{AA}\) has – every substring is a palindrome!

Our goal is to come up with an algorithm that partitions an input string into as few palindromes as possible. It will take an input string \(X=\langle x_1, x_2, \cdots, x_n\rangle\) and output the substrings in the partition.
