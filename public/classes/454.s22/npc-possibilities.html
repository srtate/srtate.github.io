Notes…

  1. Dominating set [G&J GT2] – Red from vertex cover. Basically add a new vertex for each edge and make a triangle with the original edge. DS in this graph doesn’t need to use any of the new verts, so then a DS is a VC in orig. More interesting: Still NPC on bipartite graphs, if duplicate the original G on both sides (with edges changes to be L-R and self edges added).

  2. Monochromatic Triangle [G&J GT6] – Partition edge set into two sets such that neither subgraph has a triangle. Reduce from 3SAT.

  3. Feedback Vertex Set [G&J GT7] - Is there a subset of \(k\) vertices such that every directed cycle has a vertex in the subset. Reduce from vertex cover. Just turn each undir edge in VC to a pair of dir edges. Remains NPC for undirected graphs. For simple cycle make each edge a triangle.

  4. Same as prev, but subset of edges (Feedback Arc Set) [G&J GT8] Notes that is trivially solvable in poly time for undirected graphs.

  5. Partition into Hamiltonian subgraphs [G&J GT13] (partition vertices into \(k\) sets so that each induced subgraph is Hamiltonian).

  6. Partition into forests [G&J GT14]. Undirected graph. Partition vertices into at most \(k\) disjoint sets, each of which is acyclic. Red from 3-colorability.

  7. Partition into cliques [G&J GT15]. At most \(k\) sets, each being a complete subgraph. Red from \(k\)-colorability.

  8. Induced path [G&J GT23]. At least \(k\) vertices induce a subgraph that is a simple path. Note not just longest simple path, since extraneous edges are not allowed. Red from 3SAT. (Longest path without shortcuts in given vertices. Chordless. Not nec shortest path since could be a shorter path that uses non-included vertices.)

  9. Maximum \(k\)-connected subgraph [G&J GT31] (upper-bound both subgraph vertex size and edge size). A simple version is with \(k\) fixed to 2 is still NPC, and is a pretty simple reduction from HAM-CYCLE. Note easy to solve for \(k=1\).

  10. Hamiltonian completion [G&J GT33] - min number of edges to add. Was assigned previously when I called it HAM-EXTEND.

  11. HAM-PATH without specifying endpoints.

  12. Largest common subgraph [G&J GT49] - reduce from CLIQUE (or HAM-CYCLE?). Note need to talk about isomorphic graphs for this problem to make sense.

  13. Path with forbidden pairs [G&J GT54] - endpoints s and t given, and set of pairs of vertices - path can include at most one vertex from each forbidden pair. Red from 3SAT.

  14. K-closure [G&J GT58] - Directed graph. Is there a subset of at most \(k\) vertices such that no edge goes from outside the subset to inside? Red from clique.

  15. Isomorphic spanning tree - given graph and tree, does G have a spanning tree isomorphic to T? Red from HAM-PATH.

  16. Max cut [G&J ND16] - weighted graph and lower bound k in input. Red from MAX 2-SAT. NPC even if all w=1 (so unweighted). (My note: maximizing edges across a cut is minimizing edges fully on one side of cut, so maybe red from max bipartite subgraph?)

  17. Rural postman [G&J ND27] - weighted graph, bound B, and a subset of edges. Find a circuit (not nec simple) that includes all edges in the subset, and has total weight at most B. Red from HAM-CYCLE.

  18. Exact cover by 3-sets [G&J SP2] - given 3-elem subsets, find a collection that includes each element exactly once. Red from 3DM (3D matching, which is red from 3SAT). (Could make it a “pick teams of size 3” problem.)

  19. Set packing [G&J SP3] Given collection of sets and lower bound k – does there exist at least k mutually disjoint sets?

  20. Set splitting [G&J SP4] - collection of subsets. Is there a partition of elements such that every subset has at least one element from each part.

  21. Min set cover [G&J SP5] - Collection of subsets and bound k. Can we pick k subsets that cover universe? Red from X3C in G&J. NPC even if all sets have size <= 3.

  22. Hitting set [G&J SP8] - Collection of subsets and bound k. Is there a subset of elements such that every subset in collection contains at least one of those elements. (Phrase as “representatives”?) Red from Vertex cover.

  23. Bin packing [G&J SR1] - Values, bin capacity, and bound k - divide into at most k bins? Red from PARTITION or 3-PARTITION. (Rephrase as multi-thief knapsack?)

  24. Shortest common supersequence [G&J SR8] - finite set of strings and bound k. Is there a string of length k such that each string in input is a subsequence? Red from vertex cover. NPC with alphabet size 5. Polytime solvable if just 2 strings (compute LC subsequence).

  25. Shortest common superstring [G&J SR9] - Red from Vertex cover (for cubic graphs?). NPC even with binary alphabet.

  26. Longest common subsequence [G&J SR10] - set R and bound K. Solvable for fixed K or |R|. Red from Vertex cover.

  27. Knapsack - red from PARTITION.

  28. Not-all-equal 3SAT [G&J LO3] - each clause must have at least one true and at least one false. Red from 3SAT.

  29. One-in-three 3SAT [G&J LO4] - each clause must have exactly one true. Red from 3SAT. Remains NPC even if no clause has a negated literal.

  30. Max 2-SAT [G&J LO5] - 2-CNF and bound k. Can satisfy at least k clauses? Red from 3SAT. P if k=|C|.

  31. Betweenness [G&J MS1] - triples (a,b,c) taken from set A, does there exist an ordering of A so that each b is between a and c? Red from set splitting.

  32. Clustering [G&J MS9] - distance matrix for set X, and ints k and

    1. Is there a partition of X into k sets such that max distance within each set is b? Red from 3-colorability. Still NPC for fixed K=3 and all distances in 0/1. P if k=2.

  33. k-COLOR to CLIQUE (from online forum) - Given G,k, create k*n vertex graph, edges between (u,c) and (v,d) if either uv not in G or c!=d. n CLIQUE in this graph iff k-coloring in original graph.
