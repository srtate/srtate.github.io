Additional LaTeX Tips and Information

To create great looking documents, make sure you keep “math mode” and “text mode” separate in your mind. Anything that is mathematical, including simple variable names, should always be set in math mode. For example, if you have a single letter variable x in a sentence, the LaTeX should look like “variable $x$ is defined to be…” Conversely, anything that is not mathematical should not be put in math mode. Some people incorrectly use math mode (dollar signs) to make text italic, but that is not correct. Math mode will process this text differently, and it can look very bad. Try putting $different$ into LaTeX and seeing how it looks, paying special attention to how the “f” letters are spaced. This looks bad and unprofessional, so don’t do it! Some “math words” that should be typeset in text mode rather than math mode are defined as special keywords, including “log” for logarithms and all of the trigonometric functions (listed on the Symbols section of the tutorial). Make sure you use the “backslash version” of these words so they are typeset properly! If you need to put arbitrary text within a math mode chunk of source, you can do that with \text{}.

For example, the block “\[ x^2\geq 4 \text{\ if\ } x\geq 2\]” will get set as
        x2 ≥ 4 if x ≥ 2.
Note the backslash and space at both the beginning and end of the \text parameter — this is for spacing, because otherwise math mode removes all spaces. This is a common way of adding space in LaTeX (similar to &nbsp; in HTML), and inside math mode you can use \, (with a comma after the backslash) to add a slightly smaller amount of space.

A few final tips:

  • Some common symbols have special meanings in LaTeX, so you have to “escape” them with backslashes if you want the actual symbol. For example, if you want to write that a dollar amount, like $12, you need to put a backslash before the dollar sign (like \$12) so it doesn’t switch you to math mode. Similarly, if you want to use curly braces for set notation, you need to use \{ and \}.
  • The examples in the tutorial show definite integrals, but sum and product notation works the same way. For example, the sum of the first n integers can be written using \sum_{i=1}^n i.
  • Special sets that are written with double lines in mathematics, like ℤ for the set of integers, is done by using \mathbb{Z} (or similar for rationals, natural numbers, reals, etc.). Note that you must have included the amssymb package for this to work.
  • If you want to put a line through a relation symbol, precede it with \not. For example, 1\not\eq 2 becomes 1 ≠ 2, and -1\not\in\mathbb{Z}^+ becomes  − 1 ∉ ℤ+.

That’s it – happy LaTeX’ing!
