    Show \(P\) is closed under union, concatenation, and complement.

    • Union: Let \(L_3=L_1\cup L_2\) for languages \(L_1\) and \(L_2\) that are both in \(P\). Let \(L_1\) be decided in time \(O(n^{k_1})\) and \(L_2\) be decided in time \(O(n^{k_2})\) for constants \(k_1\) and \(k_2\). Let \(k_3=\max(k_1,k_2)\), so both \(L_1\) and \(L_2\) are decided in time \(O(n^{k_3})\). To decide \(L_3\), given an input \(w\) we first run the algorithm to decide whether \(w\in L_1\) – if so we accept, because \(w\in L_3\). If \(w\not\in L_1\) then we run the decision algorithm for \(L_2\) and accept if and only if \(w\in L_2\) (reject otherwise). This clearly decides \(L_3\) and runs in time \(O(n^{k_3})\), so \(L_3\in P\). Therefore \(P\) is closed under union.

    • Concatenation: Given any two languages \(L_1,L_2\in P\), the concatenation language \(L_3\) is

      \[ L_3=\{xy\,|\, x\in L_1\text{ and }y\in L_2\} \]

      Given a string \(z\) of length \(n\), we can test whether \(z\in L_3\) by trying all possible “dividing lines” for the concatenation. In other words, every way of breaking up \(z\) into strings \(x\) and \(y\) with \(z=xy\). For each possibility, we check whether \(x\in L_1\) and \(y\in L_2\). If the deciders for \(L_1\) and \(L_2\) run in time \(O(n^{k_1})\) and \(O(n^{k_2})\), respectively, and we let \(k_3=\max(k_1,k_2)\), then each possible dividing line can be tested in \(O(n^{k_3})\) time. There are \(n\) possible “dividing lines”, so total running time to decide \(L_3\) is \(O(n\cdot n^{k_3})=O(n^{k_3+1})\), so \(L_3\in P\). Therefore, \(P\) is closed under concatenation.

    • Complement: Complement is the easiest one. Given an that decides some language \(L\in P\) in some polynomial amount of time, say \(O(n^k)\), we run the same algorithm but flip the outcome: accept when we would have rejected, and reject when we would have accepted. This decides \(\overline{L}\) in \(O(n^k)\) time, so \(\overline{L}\in P\). Therefore \(P\) is closed under complement.

(Textbook, Exercise 7.9) [How to “grade” your work (8 points): The key to this problem, and really the only way to approach it, is the fact that you can brute-force this problem since there are only \(O(n^3)\) possible triangles, and each possible triangle is easy to check. You do need to have both parts here though: bound the number of triangles by \(O(n^3)\) (5 points – 3 points for the less-useful vague statement about a “polynomial number”, since you really should give the \(O(n^3)\) value); then describe how each can be checked in polynomial time (2 points – it’s ok to be vague with “polynomial” here, since the exact polynomial will depend on the machine model, like TM versus other computer). You should conclude that \(\textit{TRIANGLE}\in P\) (1 point). ]

    The exercise asks to show that the following language is in \(P\):

    \[ \textit{TRIANGLE} = \{\langle G\rangle\,|\, G \text{ contains a triangle} \}\]

    A triangle is a set of 3 vertices that are all connected to each other. To decide \(\textit{TRIANGLE}\) you can just check every set of three vertices to see if they form a triangle. Given a set of three vertices, it’s trivial to check if they form a triangle (there are just three edges to check for). Since there are only \({n \choose 3}=O(n^3)\) possible triangles, this is clearly polynomial time. Therefore, \(\textit{TRIANGLE}\in P\).

    The problem asks to show that the following language is in \(NP\):

    \[ \textit{FACTOR} = \{ \langle x,k\rangle\,|\, x \text{ is an integer that is divisible by an integer } y\in[2,k] \} \]

    For any \(\langle x,k\rangle\in\) \(\textit{FACTOR}\), a certificate is just a number \(a\leq k\) that is greater than 1 and is a factor of \(n\) (in other words, \(a\) is a non-trivial factor \(\leq k\)). Since this is the definition of \(\textit{FACTOR}\), such a value \(a\) exists if and only if \(\langle x,k\rangle\in\textit{FACTOR}\). The verifier is defined as follows:

    \(V=\) "On input \(\langle\langle x,k\rangle,a\rangle\):

    1. Test whether \(a\in [2,k]\).

    2. Test whether \(a\,|\,n\) (i.e., \(a\) divides evenly into \(n\)).

    3. If both pass, accept; otherwise, reject."

    \(V\) is clearly polynomial time, and accepts if and only if \(a\) is a valid certificate (which exists if and only if \(\langle x,k\rangle\in\textit{FACTOR}\)). Therefore, \(\textit{FACTOR}\in NP\).
