    No, \(A\) is not necessarily regular. Consider the following counter-example.

    \[ A = \{ \texttt{0}^n\texttt{1}^n\,|\, n\geq 0 \} \] \[ B = \{ \texttt{0}^*\texttt{1}^*\,|\, n\geq 0 \} \]

    Notice that \(B\) is regular, but \(A\) is not. We give a reduction to show that \(A\leq_{m} B\). The only thing that is important here is that \(A\) is decidable on its own, so the reduction \(f\) takes an input \(w\), decides if it’s in \(A\), and then outputs a string that’s in \(B\) if it is, and one that’s not if it isn’t. Here’s such a reduction function:

    \[ f(w) = \left\{ \begin{array}{ll} \texttt{01} & \text{if } w\in A;\\ \texttt{10} & \text{otherwise.} \end{array} \right. \]

    \(f\) is obviously computable, and \(w\in A\) if and only if \(f(w)\in B\). Therefore, \(f\) is a valid many-to-one reduction, showing that \(A\leq_{m} B\).

    The bottom line here is that there are decidable regular languages and non-regular languages, and any two non-trivial decidable languages can be reduced to each other. To summarize, we have languages \(A\) and \(B\) with \(A\leq_{m} B\), where \(B\) is regular and \(A\) is not. This counter-examples shows that the answer to the question is “no.”

    We will show the following language is undecidable:

    \[ CFL_{TM}=\{\langle M\rangle\,|\,L(M) \text{ is a context-free language}\} \]

    Assume for the sake of contradiction that \(CFL_{TM}\) is decidable, and let \(R\) be a TM that decides this language. We construct TM \(S\) that uses \(R\) to decide \(A_{TM}\).

    \(S=\) "On input \(\langle M,w\rangle\), where \(M\) is a TM and \(w\) is a string:

    1. Construct the following TM \(M_2\).

      \(M_2=\) "On input \(x\):

      1. If \(x\) has the form \(\texttt{0}^n\texttt{1}^n\texttt{0}^n\), accept.

      2. If \(x\) does not have this form, run \(M\) on input \(w\) and accept if \(M\) accepts \(w\)."

    2. Run \(R\) on input \(\langle M_2\rangle\).

    3. If \(R\) accepts, accept; if \(R\) rejects, reject."

    Consider the behavior of \(S\):

    • If \(\langle M,w\rangle\in A_{TM}\) then \(M\) accepts \(w\), and so on any input \(M_2\) accepts (if not in step 1, then it will in step 2). So when \(M\) accepts \(w\), we have \(L(M_2)=\Sigma^*\) (all strings), and so \(L(M_2)\) is a context-free language. This means that \(R\) will accept \(M_2\), and \(S\) will accept \(\langle M,w\rangle\).

    • If \(\langle M,w\rangle\not\in A_{TM}\), then \(M\) does not accept \(w\), and so \(M_2\) only accepts inputs \(x\) in step 1. Therefore, \(L(M_2)=\{\texttt{0}^n\texttt{1}^n\texttt{0}\,|\,n\geq 0\}\). This is not a context-free language, so \(R\) rejects \(M_2\) and then \(S\) rejects \(\langle M,w\rangle\).

    This means that \(S\) decides \(A_{TM}\), which is impossible and gives a contradiction. Therefore, \(CFL_{TM}\) is undecidable.
