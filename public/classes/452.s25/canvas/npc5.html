Creating a proof – NP membership

In the previous sections, we talked about what components needed to be in an NP-completeness proof, and how to evaluate whether you have established all of the points that need to be made. Now we look at how to create your proof from scratch – not just what needs to be there, but how you come up with the right ideas, and common mistakes students make when writing a proof.

Proving membership in NP

The first component of the NP-completeness proof is the proof that the language is in NP. Using the certificate/verifier formulation of NP, if we want to show that a language \(L\) is in \(NP\), we need to show that an input \(x\) has a polynomial-sized certificates if and only if \(x\in L\), where the certificate can be verified in polynomial time.

Here are some examples of problems/languages that we’ll use in this discussion. Many of these are graph problems, and I assume you know the basic graph terminology – if not, see explanations here.

SAT \(=\{\langle\phi\rangle : \phi\) is a Boolean formula for which some assignment of Boolean values to the variables makes \(\phi\) true\(\}\)

CLIQUE \(=\{\langle G,k\rangle : G\) is an undirected graph that contains a clique of size \(k\}\)

VERTEX-COVER \(=\{\langle G,k\rangle : G\) is an undirected graph that contains a vertex cover of size \(k\}\)

HAMCYCLE \(=\{\langle G\rangle : G\) is an undirected graph that contains a Hamiltonian cycle\(\}\)

HAMPATH \(=\{\langle G,s,t\rangle :\) there is a Hamiltonian path from \(s\) to \(t\) in graph \(G\}\)

ANY-HAMPATH \(=\{\langle G\rangle :\) there is a Hamiltonian path in graph \(G\}\)

HAM-EXTEND \(=\{\langle G,k\rangle : k\geq 1\) is an integer and there exists a set \(E'\) of edges such that \(|E'|\leq k\) and \(G'=(V,E\cup E')\) is Hamiltonian \(\}\)
